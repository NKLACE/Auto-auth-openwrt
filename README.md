
##📌 项目介绍

auto-auth 是一个 河南工学院校园网自动认证脚本，适用于在 OpenWrt 等 Linux 环境下自动完成校园网登录，解决早晨来电后需要手动认证的问题。

##🛠 获取认证参数（账号信息填写方式）

打开校园网认证页面

按 F12 打开开发者工具

切换到 Network（网络） 面板（WiFi 图标）

勾选 Preserve log（保留日志）

正常输入校园网账号、密码，并选择运营商完成一次认证

在请求列表中找到名为 quickdo 的请求

找到 包含账号和密码参数的请求链接，并复制该链接

将 该链接 + 本脚本 一并发送给任意 AI，让其帮你填写脚本中的参数即可


##🚀 使用方法

将脚本上传到 OpenWrt 的任意目录，例如：

/root/autologin.sh


赋予脚本执行权限：

chmod +x autologin.sh


编辑开机启动文件：

vim /etc/rc.local


在 exit 0 之前添加以下内容：

sleep 10
/你的路径/autologin.sh


保存并退出：

按 Esc

输入 :wq

回车

重启设备即可生效 🎉
